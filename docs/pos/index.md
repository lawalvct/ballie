# Point of Sale

Ballie's POS module provides a fast, intuitive checkout experience. Open a cash register session, process sales, accept multiple payment methods, and print receipts.

## Features

- **Cash Register** — Open and close register sessions with cash tracking
- **Quick Checkout** — Scan or search products, apply discounts
- **Split Payments** — Accept multiple payment methods per transaction
- **Receipts** — Print or email receipts to customers

## Quick Start

1. Set up a [cash register](/pos/cash-register) in Settings
2. Open a register session with your opening cash balance
3. Start [making sales](/pos/making-sales)
