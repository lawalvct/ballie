# Managing Projects

Create projects to organize work, track progress, and manage deliverables.

## Creating a Project

1. Navigate to **Projects**
2. Click **+ New Project**
3. Fill in:
   - **Project Name** — A descriptive title
   - **Client** (optional) — Link to a customer
   - **Start Date** and **End Date**
   - **Budget** (optional) — Estimated project cost
   - **Description** — Project scope and details
4. Click **Save**

## Tasks and Milestones

- Add tasks within a project to break down work
- Set milestones for key deliverables
- Assign team members to tasks
- Track completion status

## Attachments

Upload files and documents to projects for easy reference.
