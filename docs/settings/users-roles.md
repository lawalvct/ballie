# Users & Roles

Control who can access your Ballie workspace and what they can do with role-based permissions.

## Inviting Users

1. Navigate to **Settings → Users**
2. Click **+ Invite User**
3. Enter their **Email** and select a **Role**
4. Click **Send Invite**

## Creating Roles

1. Navigate to **Settings → Roles**
2. Click **+ New Role**
3. Enter the **Role Name**
4. Select permissions for each module (View, Create, Edit, Delete)
5. Click **Save**

## Default Roles

| Role | Access Level |
|------|-------------|
| Admin | Full access to all modules |
| Accountant | Accounting, reports, and banking |
| Sales | Invoices, POS, CRM |
| HR Manager | Payroll & HR module |
| Viewer | Read-only access |

::: warning
Only Admin users can manage roles and invite new users.
:::
