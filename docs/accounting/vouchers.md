# Vouchers

Journal vouchers allow you to make manual accounting entries. Use them for adjustments, corrections, and transactions that don't originate from invoices or expenses.

## Creating a Voucher

1. Navigate to **Accounting → Vouchers**
2. Click **+ New Voucher**
3. Select the **Voucher Type** (Journal, Payment, Receipt, etc.)
4. Add debit and credit entries — totals must balance
5. Add a **Narration** describing the purpose
6. Click **Save**

## Posting and Unposting

- **Post** — Finalizes the voucher and updates ledger balances
- **Unpost** — Reverses the ledger effect (for corrections)

::: warning
Only unpost a voucher if you need to correct an error. Frequent unposting can make your audit trail harder to follow.
:::
