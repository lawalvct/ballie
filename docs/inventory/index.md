# Inventory

Ballie's Inventory module helps you manage products, track stock levels, handle stock movements, and conduct physical audits.

## Features

- **Products** — Add and manage products with images, pricing, and SKU
- **Categories & Units** — Organize products by category and unit of measurement
- **Stock Management** — Track stock movements, transfers, and journals
- **Physical Stock Audit** — Conduct physical counts and reconcile with system records

## Quick Start

1. Set up your [product categories](/inventory/categories)
2. [Add your products](/inventory/products) with pricing and initial stock
3. Use [stock journals](/inventory/stock-management) for stock adjustments
4. Run [physical stock audits](/inventory/physical-stock) periodically
