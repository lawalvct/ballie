# Physical Stock Audit

Physical stock audits help you verify that your actual inventory matches the system records. Conduct regular audits to catch discrepancies.

## Conducting an Audit

1. Navigate to **Inventory → Physical Stock**
2. Click **+ New Audit**
3. Select the products to audit (or select all)
4. Enter the **Physical Count** for each product
5. Ballie calculates the **Variance** (difference between system and physical count)
6. Review and click **Submit**

## Resolving Variances

When physical counts differ from system records:
- **Positive variance** — More stock than expected (adjust up)
- **Negative variance** — Less stock than expected (adjust down)

Approved audits automatically create stock adjustment journals.
