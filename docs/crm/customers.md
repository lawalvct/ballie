# Customers

Manage all your customer records in one place. Add customers individually, import them in bulk, set opening balances, and track all interactions.

## Adding a Customer

1. Navigate to **CRM → Customers**
2. Click **+ New Customer**
3. Fill in:
   - **Customer Name**
   - **Email** and **Phone**
   - **Address** details
   - **Tax ID** (optional)
   - **Opening Balance** (if migrating from another system)
4. Click **Save**

## Bulk Import

Import multiple customers from a spreadsheet:
1. Click **Import Customers**
2. Download the CSV template
3. Fill in your customer data
4. Upload the completed file
5. Map columns and confirm

## Opening Balances

Set opening balances when migrating to Ballie to maintain accurate accounts receivable records.
