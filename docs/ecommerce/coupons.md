# Coupons

Create discount codes for your online store with flexible rules and usage limits.

## Creating a Coupon

1. Navigate to **E-Commerce → Coupons**
2. Click **+ New Coupon**
3. Configure:
   - **Coupon Code** — The code customers enter
   - **Discount Type** — Percentage or fixed amount
   - **Discount Value** — How much off
   - **Minimum Order Amount** (optional)
   - **Usage Limit** — Maximum number of uses
   - **Expiry Date**
4. Click **Save**
