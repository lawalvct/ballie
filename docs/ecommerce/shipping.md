# Shipping

Set up shipping methods, rates, and delivery zones for your online store.

## Configuring Shipping

1. Navigate to **E-Commerce → Shipping**
2. Add shipping methods (e.g., Standard, Express, Pickup)
3. Set rates per zone or flat rates
4. Enable or disable methods as needed
