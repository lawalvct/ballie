# Store Setup

Configure your online store's appearance, branding, and settings.

## Configuration

1. Navigate to **E-Commerce → Settings**
2. Set up:
   - **Store Name** and **Description**
   - **Logo** and **Banner Images**
   - **Custom Domain** (optional)
   - **Currency** and **Tax Settings**
   - **Contact Information**
