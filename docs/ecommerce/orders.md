# Orders

Track and fulfill customer orders from your online store.

## Order Management

1. Navigate to **E-Commerce → Orders**
2. View all orders with status indicators
3. Click an order to see details
4. Update the order status as you fulfill it

## Order Statuses

| Status | Description |
|--------|-------------|
| Pending | New order, awaiting processing |
| Processing | Being prepared for shipment |
| Shipped | Dispatched to customer |
| Delivered | Successfully received |
| Cancelled | Order cancelled |
