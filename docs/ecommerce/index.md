# E-Commerce

Ballie's E-Commerce module lets you set up an online storefront, manage orders, create discount coupons, and configure shipping.

## Features

- **Store Setup** — Branding, domain, and storefront configuration
- **Orders** — Order management and fulfillment tracking
- **Coupons** — Discount codes with usage limits
- **Shipping** — Shipping methods, rates, and zones

## Quick Start

1. [Configure your store](/ecommerce/store-setup) branding and settings
2. Add products through the [Inventory module](/inventory/products)
3. [Manage incoming orders](/ecommerce/orders)
