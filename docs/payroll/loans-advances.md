# Loans & Advances

Manage employee loans with automatic repayment schedules that integrate with payroll processing.

## Creating a Loan

1. Navigate to **Payroll → Loans**
2. Click **+ New Loan**
3. Enter:
   - **Employee** — Select the employee
   - **Loan Amount** — Total loan value
   - **Interest Rate** (if applicable)
   - **Repayment Period** — Number of months
   - **Start Date** — When repayments begin
4. Click **Save**

## Repayment

- Ballie automatically generates a repayment schedule
- Monthly deductions are included in payroll processing
- Track remaining balance and payment history

## Salary Advances

For one-time advances:
1. Navigate to **Payroll → Salary Advance**
2. Create an advance request
3. The advance is deducted from the next payroll
