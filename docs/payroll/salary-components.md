# Salary Components

Define the building blocks of your payroll — earnings, deductions, allowances, and tax brackets.

## Component Types

| Type | Description | Example |
|------|-------------|---------|
| Earnings | Additions to gross pay | Basic Salary, Housing Allowance, Transport |
| Deductions | Subtractions from gross pay | Pension, Tax, Loan Repayment |
| Statutory | Government-mandated deductions | PAYE, Pension (Employee), NHF |

## Creating a Component

1. Navigate to **Payroll → Salary Components**
2. Click **+ New Component**
3. Set the type (Earning/Deduction), name, and calculation method
4. Choose whether it's a fixed amount or percentage
5. Click **Save**

## Assigning to Employees

Components can be assigned:
- **Globally** — Applied to all employees
- **Per Employee** — Custom amounts on individual salary setup
