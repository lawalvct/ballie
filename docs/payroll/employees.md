# Employees

Manage complete employee records including personal information, employment details, salary setup, and documents.

## Adding an Employee

1. Navigate to **Payroll → Employees**
2. Click **+ New Employee**
3. Fill in:
   - **Personal Details** — Name, email, phone, date of birth, gender
   - **Employment Details** — Employee ID, department, position, hire date
   - **Bank Details** — Bank name, account number (for salary payment)
   - **Statutory Details** — Tax ID (TIN), pension PFA, NHF number
4. Click **Save**

## Employee Profile

Each employee profile includes:
- Personal and contact information
- Employment history
- Salary details and history
- Documents (contracts, certificates, ID copies)
- Payslip history
