# Mobile App

The Ballie mobile app lets you manage your business from anywhere. Available for Android and iOS with offline support.

## Features

- Access key modules on mobile (Dashboard, Invoices, CRM, POS, Inventory)
- Offline mode — work without internet, sync when connected
- Push notifications for important events
- QR code attendance check-in
