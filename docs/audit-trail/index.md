# Audit Trail

Ballie's audit trail records every significant action in your workspace — who did what, when, and what changed.

## What's Tracked

- Record creation, updates, and deletions
- User login and logout events
- Financial transactions (invoices, vouchers, payments)
- Settings changes
- User and role modifications
