# Viewing Audits

Search and filter the audit trail to find specific actions and changes.

## Accessing the Audit Log

1. Navigate to **Audit Trail** from the sidebar
2. Browse the chronological list of events

## Filtering

Filter audits by:
- **Date Range** — Specify start and end dates
- **User** — See actions by a specific user
- **Module** — Filter by Accounting, Inventory, CRM, etc.
- **Action Type** — Create, Update, Delete

## Exporting

Export audit logs for compliance or review:
1. Apply your desired filters
2. Click **Export**
3. Download as CSV or PDF
