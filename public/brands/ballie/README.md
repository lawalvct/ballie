# Ballie Brand Assets

Place brand-specific assets here:

- `logo.png` - Primary logo (recommended: 200x60px or similar aspect ratio)
- `logo-white.png` - White/inverted logo for dark backgrounds
- `favicon.ico` - Favicon (32x32px)
- `favicon-16x16.png` - Small favicon
- `favicon-32x32.png` - Standard favicon
- `apple-touch-icon.png` - Apple touch icon (180x180px)
- `og-image.png` - Open Graph image for social sharing (1200x630px)

## Current Brand Colors

- Primary: #4F46E5 (Indigo)
- Secondary: #7C3AED (Purple)
- Gradient From: #3c2c64
- Gradient Via: #4a3570
- Gradient To: #614c80

## Usage

These assets are automatically loaded by the `BrandService` class when the brand is detected.
